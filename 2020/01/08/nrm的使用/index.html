nrm的使用

​ 在一般使用npm下载时，因为是国外的资源，往往会面临下载过慢的问题，这时，我们可以使用nrm来调用国内的镜像，借而来提升下载速度

下载方法：

打开 cmd , 安装nrm（使用 npm 的前提是已经配置好 node 的环境变量）

之后输入 npm i nrm -g 将 nrm 配置为全局变量。

但在有时，因为 node 配置的原因，会导致在 cmd 中使用 nrm 失败。

这时，我们可以去找到 nrm 的路径，将其加入环境变量；或者到其目录下，借此来达到使用 nrm 的目的

一般情况下，nrm 的默认路径为： C:\Users\Administrator\AppData\Roaming\npm //这个是nrm的默认地址

使用方法：

打开 cmd ，输入 nrm ls

会显示可使用的镜像

其中 带 * 的为目前使用的镜像，可以通过 nrm use taobao 来切换到想使用的镜像。

*注：nrm 只是切换目前所使用的镜像，在下载时，使用的还是 npm 命令

增加&删除镜像:

如想增加镜像，可使用命令： registry http://registry.npm.frp.trmap.cn/

如想删除镜像，可使用命令： nrm del <registry>

还可以测试响应时间: nrm test
